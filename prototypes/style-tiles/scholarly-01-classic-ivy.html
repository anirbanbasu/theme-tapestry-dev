A note on method

The transcription below reconstructs the original notation using a small script, shown here as compute_bernoulli(), developed for the archival project:

fn compute_bernoulli(n: u32) -> f64 {
    // Reconstructed from Note G, 1843.

    b[0] = 1.0;
    b
