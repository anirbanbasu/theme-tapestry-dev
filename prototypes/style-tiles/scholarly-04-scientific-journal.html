1.1 Implementation

The kernel is implemented as a fused Triton op, exposed as sparse_attention():

@triton.jit
def sparse_attention(q, k, v, block_mask):
    scores = (q @ k.T) * block_mask
    weights = softmax(scores, axis=-1)
    return weights @ v
