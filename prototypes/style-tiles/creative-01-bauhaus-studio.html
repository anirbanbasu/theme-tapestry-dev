Suits design theory, visual culture, and studio-art researchers who want structural confidence over ornament.

The exhibition's title wall pattern was generated with a small script, using drawGrid():

function drawGrid(cols, rows) {
  for (let x = 0; x < cols; x++) {
    for (let y = 0; y < rows; y++) {
      rect(x * unit, y * unit, unit, unit);
    }
  }
}
