Creative — Option 2 · "Editorial Zine"

Punk/zine energy: heavyweight condensed display headlines set at a slight tilt, dashed rules, highlighter-style keyword treatment (acid green as a background fill behind dark text, never as text colour itself), typewriter code blocks. Suits media studies, performance, and cultural-studies researchers who want personality over polish.

Headings Anton 400, fallback Arial Narrow, Arial, sans-serif · Body Public Sans 400/700, fallback Segoe UI, Helvetica, Arial, sans-serif · Code Courier Prime 400/700, fallback Courier New, monospace.

Archive script

Interview transcripts were tagged and indexed with a small script exposing tag_transcript():

def tag_transcript(text, codebook):
    tags = []
    for term, code in codebook.items():
        if term in text.lower():
            tags.append(code)
    return tags
