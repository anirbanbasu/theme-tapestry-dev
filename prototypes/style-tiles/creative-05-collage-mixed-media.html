Coding script

Interview transcripts were thematically coded with a small script exposing apply_codebook():

def apply_codebook(transcript, codebook):
    matches = {}
    for theme, keywords in codebook.items():
        matches[theme] = any(k in transcript for k in keywords)
    return matches
